# sdet-automation-framework
SDET Automation Framework using Selenium, Java, TestNG, and Rest Assured.

The project showcases both UI and API automation with a CI-ready structure.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Rest Assured
- Maven
- GitHub

## Framework Highlights
- UI automation using Selenium WebDriver
- API automation using Rest Assured
- TestNG-based execution
- Maven build management
- Scalable and maintainable framework design

## Project Structure
- base      → WebDriver setup and teardown
- tests.ui → UI automation tests
- tests.api → API automation tests
- utils     → Common utilities
